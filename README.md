# Portfolio BUT Réseaux & Télécommunications — squelette

Ce dossier contient un squelette de site portfolio conforme aux attendus :
- 4 espaces principaux : Accueil, Présentation & projection, SAÉ (1 page par SAÉ), Bilans (1 page par compétence et par semestre)
- chaque page SAÉ suit la structure : description/objectifs → AC → tâches → traces → autoévaluation

## Démarrer
- Ouvre `index.html` dans un navigateur (les chemins sont relatifs, donc ça marche en double-cliquant).

## Personnalisation rapide
1. Remplace `Ton Nom` et ajoute ta photo : `assets/images/photo-pro.jpg`
2. Ajoute ton CV : `assets/documents/cv.pdf`
3. Sur chaque page SAÉ : ajoute 1–2 traces individuelles et commente-les.
4. Sur chaque page bilan : classe les AC (maîtrisés / en cours / à renforcer) + objectifs.

Astuce : garde un ton analytique et professionnel (pas seulement descriptif).
